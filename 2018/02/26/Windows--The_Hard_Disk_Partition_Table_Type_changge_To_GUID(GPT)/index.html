如何将硬盘分区表转换为GUID(GPT)分区表

如果不知道自己的硬盘分区表类型可以参看 Windows–如何查看硬盘分区表类型

1、使用PE

  • 制作U盘PE启动盘
  • 进入PE
  • 在PE里打开磁盘精灵 （DG）
  • 右键磁盘 菜单中有转换为GUID分区表

2、在安装Win的时候

  • 制作UEFI启动盘
  • 进入UEFI安装的界面
  • shift + F10 （打开终端）
  • 输入diskpart 回车 （进入diskpart）
  • list disk (显示磁盘列表，注意这里你只能用大小来区分你的硬盘了)
  • sel disk 0 (这个编号你自己看着你的来)
  • clean （清除数据）（可选）
  • convert gpt (转换为GPT分区表，好像是会清除磁盘上数据的)

同理，GPT转换至MBR上述方法同样适用
